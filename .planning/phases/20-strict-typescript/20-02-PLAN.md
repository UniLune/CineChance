<objective>
Eliminate all `any` type usages and unused variable violations across the codebase to achieve strict TypeScript compliance.

**Purpose:** `any` defeats TypeScript's type safety. This task replaces all `any` with explicit types or `unknown` where appropriate, and removes unused variables to meet production lint standards.

**Output:** Codebase with zero `any` violations and zero unused variable warnings. All TypeScript files compile under strict mode.

**Strategy:**
- Replace `any` with specific interfaces for known data shapes (TMDB responses, API payloads, Prisma models)
- Use `unknown` for truly dynamic data with runtime type guards
- For external library data, create local type definitions
- Remove unused imports/parameters; prefix intentionally unused with `_`
</objective>
