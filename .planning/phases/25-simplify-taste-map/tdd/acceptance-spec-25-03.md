# Acceptance Spec: Phase 25-03 — Remove Chart Visualizations

## User Story

Как пользователь, я хочу видеть на странице «Карта вкуса» только текстовые метрики и блок похожих пользователей, без графиков жанров и круговой диаграммы оценок.

## Scenarios

### Scenario 1: Страница вкуса без графиков
**Given** пользователь перешёл на `/profile/taste-map/`
**And** у него есть данные вкуса (genreProfile, ratingDistribution)
**When** страница загружается
**Then** он видит 4 карточки сверху (Средний рейтинг, Положительные оценки, Консистентность, Разнообразие)
**And** он видит блок «Метрики профиля» с 4 карточками
**And** он видит блок «Поведенческий профиль» с 3 карточками
**And** он видит блок «Ваши близнецы вкуса» (TwinTasters)
**And** он НЕ видит блок «Профиль жанров» (горизонтальная столбчатая диаграмма)
**And** он НЕ видит блок «Распределение оценок» (круговая диаграмма)

### Scenario 2: Пустое состояние без изменений
**Given** пользователь перешёл на `/profile/taste-map/`
**And** у него нет данных вкуса (genreProfile пуст)
**When** страница загружается
**Then** он видит заглушку «Карта вкуса пуста» с кнопкой «Добавить фильмы»
**And** никакие графики не рендерятся

### Scenario 3: Компонент TasteMapClient не импортирует recharts
**Given** код компонента TasteMapClient.tsx
**When** анализируются импорты
**Then** импорт из `recharts` отсутствует
**And** переменные genreData, ratingData, typeData, COLORS отсутствуют
